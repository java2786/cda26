# 30 Python Practice Problems  
## For Beginners — After Learning: Data Types, Variables, Print, Input, If-Else  
  
---  
  
**Instructions for Students:**  
- Solve each problem using only what you have learned so far.  
- No loops, no functions, no lists — just variables, data types, print, input, and if-else.  
- Read each problem carefully before writing code.  
- Test your solution with different inputs.

## Tips for Students  
  
1. Always convert input using int() or float() before comparing numbers.  
2. Test your code with at least 3 different inputs — including edge cases.  
3. If your output is wrong, add a print statement to check the value of your variable.  
4. Do not jump to the answer — read the problem twice before writing even one line.
