# Class Objects: Basic Problem Statement
	Focus is on creating a simple representation of a Library system where we can manage Books. The problem can be broken down into the following components:

## Book Attributes: 
Each book has specific attributes, such as title, author, and year published.
## Library Management: 
The library can contain multiple books. We will need methods to add books, remove books, and list all books in the library.

## Problem Statement

### Define a Book Class:
	Each book should have:
        title (string)
        author (string)
        year published (integer)

### Define a Library Class:
	The library should:
		Hold a collection of books.
		Have methods to:
			add_book(book): Add a new book to the library.
			remove_book(book): Remove a book from the library.
			list_books(): List all books currently in the library.

### Example Outline
	The Book class will handle creating and managing the properties of books.
	The Library class will handle a collection of Book instances and provide functionalities for managing them.
