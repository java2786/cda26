## Assignments
### Assignment 1:
- Ask user to enter two float numbers. 
- Multiply them and show result. 
- If user enters text, repeat until valid numbers.

### Assignment 2:
- Build a calculator that accepts two numbers and an operator (+, -, *, /). - Validate all inputs using try/except.

### Assignment 3 (Advanced):
Create a voting app: 
- Ask user age 
- If age < 0 or not a number, prompt again 
- If age < 18, say not eligible; 
- else say eligible
